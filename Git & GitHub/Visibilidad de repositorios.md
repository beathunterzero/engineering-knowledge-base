GitHub permite que un repositorio sea **público** o **privado**, y lo mejor es que **puedes cambiarlo en cualquier momento**, incluso después de haberlo creado.
# Cambiar un repositorio público a privado

1. Entra al repositorio en GitHub
2. Ve a **Settings**
3. Baja hasta la sección **Danger Zone**
4. Busca la opción: **“Change repository visibility”**
5. Selecciona **Make private**
6. Confirma escribiendo el nombre del repositorio
## Consideraciones:

- Si el repo es público y lo vuelves privado, **los forks públicos seguirán existiendo**.
- GitHub te advertirá si hay dependencias o acciones públicas.

# Cambiar un repositorio privado a público

1. Entra al repositorio
2. Ve a **Settings**
3. Baja hasta **Danger Zone**
4. Selecciona **Make public**
5. Confirma escribiendo el nombre del repo

## Consideraciones:

- Al hacerlo público, cualquier persona podrá verlo y clonarlo.
- Ideal para documentación técnica, portafolios o proyectos open-source.
